📂 FLIGHT LOG ANALYSIS (HIGH THROTTLE)
No file selected
Select a Betaflight blackbox CSV file and click ANALYZE.

STEP 1: CALCULATE BASELINE PIDs

  • Enter motor KV, battery voltage, prop size, weight and flying style.
  • Click CALCULATE PIDs to get conservative baseline values.
  • Click APPLY PIDs TO FC to write them directly to the PID Tuning tab, or use COPY ALL VALUES to copy them to the clipboard.

STEP 2: FLY THE TEST PATTERN

Enable Blackbox before flying:

  • Configuration tab → Blackbox → Enable, Device = SD Card, Rate = 1/2
  • Enable: Gyro, Accelerometer, Motor, PID, RC Command, Setpoint, Unfiltered Gyro

Flight pattern:

  • Level mode: Full left → right roll for 5 s, pause, full forward → back pitch for 5 s
  • Acro mode: Slowly ramp throttle 0 → 100 % over 2 s, then land

STEP 3: ANALYZE THE LOG

  • Pull the SD card, open the .BBL file in Betaflight Blackbox Explorer
  • Export as CSV (File → Export CSV)
  • Load the CSV in the ANALYZER tab and click ANALYZE

INTERPRETING RESULTS

  • EXCELLENT / CLEAN – filters are well-tuned, no changes needed
  • GOOD – minor adjustments may help
  • FAIR – lower Gyro Lowpass 2 by ~30 Hz, re-test
  • WEAK – lower by ~50 Hz, consider adding a Notch filter
  • VERY WEAK – aggressive filter reduction needed; check for mechanical vibration

FILTER RECOMMENDATION NOTE

The Gyro Lowpass 2 value shown in the Calculator is a starting point based on prop size. Use the Analyzer results to fine-tune after flying.
